# Interface Definitions

本文件定义系统各节点之间的数据交换格式。

修改接口前必须同步更新本文件。


## 节点

- Raspberry Pi (ROS2 Core)
- ESP32-P4 (Remote Terminal)
- PC Ubuntu (Visualization)


---

## Raspberry Pi → ESP32-P4

发送周期：10–20 Hz

字段：

- mode: string
- distance_mm: int
- velocity: float
- system_state: string

说明：

distance_mm 为毫米波最近反射峰距离。


---

## ESP32-P4 → Raspberry Pi

事件触发：

- joystick_x: int (-100 ~ 100)
- joystick_y: int (-100 ~ 100)
- emergency_stop: bool


---

## Raspberry Pi → PC Ubuntu

ROS2 Topics：

- /scan
- /map
- /tf


---
