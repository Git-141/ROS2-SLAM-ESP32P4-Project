# ESP32-P4 Remote Legacy Archive

这里保存旧的 ESP32-P4 无线触控遥控器路线资料。

这些资料已经不再属于当前主线。当前主线改为：

- Raspberry Pi 5：ROS2 核心计算。
- Raspberry Pi 4B + 同款 10.1 寸屏：触控操作终端。

本目录保留的价值：

- ESP32-P4 + 10.1 寸屏显示和触摸调试经验。
- LVGL UI 原型经验。
- 中文 / 日文显示、FreeType、SD 卡字体、位图字库实验记录。
- 旧的 WiFi 文件传输和 ESP32-C6 hosted WiFi 联调记录。

除非决定恢复 ESP32-P4 遥控器路线，否则不要把这里的文档当作当前实现依据。
